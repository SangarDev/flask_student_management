# Flask Student Management System

A simple web application to manage students using **Python, Flask, and SQLite**. Users can **add, edit, view, and delete student records** through a user-friendly web interface.

---

## Features
- Add new student records with name, age, and course.
- Edit existing student information.
- Delete student records.
- View all students in a responsive table.
- Simple and clean UI with CSS styling.

---

## Technologies Used
- **Python 3.14**
- **Flask** (Web Framework)
- **Flask-SQLAlchemy** (Database ORM)
- **SQLite** (Database)
- **HTML / CSS / Jinja2** (Frontend Templates)
